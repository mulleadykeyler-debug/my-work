# 设计师作品集网站 - 白色极简风格

这是一个采用白色极简设计风格的设计师作品集网站，强调精致的线条元素、几何构图和层次交互，创造出极具现代感的视觉体验。

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- GSAP 动画库

## 主要特性

### 1. 滚动差分效果

实现了不同屏幕间的滚动速度差异，创造出深度感和层次体验：

- 前覆盖后模式：前屏z-index大于后屏，后屏从前屏下方滑出
- 使用GSAP ScrollTrigger精确控制过渡效果

### 2. 线条元素系统

通过精确排布的线条引导视觉流动，强化几何感和空间结构：

- 水平对齐线与内容精确对齐
- 折角线用于划分区域
- 动态线条元素随滚动变化

### 3. 高密度布局

精确而紧凑的内容布局，最大化信息传递效率：

- 使用12列网格系统组织内容
- 清晰的内容分区和层次结构
- 适度的元素重叠创造深度感

### 4. 自定义鼠标交互

增强用户体验，强化设计感：

- 自定义跟随光标效果
- 元素悬停时的微妙变化
- 磁性效果和动态反馈

### 5. 黑白对比色彩方案

以白色为基调，配合精确的色彩对比元素：

- 黑白高对比色调为基础
- 少量关键位置使用强调色
- 不同层级使用不同的色彩饱和度

## 功能模块

1. **响应式导航栏**：固定顶部，支持移动端菜单
2. **Hero区域**：简洁的设计师名称和职业定位，配合几何线条装饰
3. **作品展示区**：网格布局展示设计作品，支持分类筛选
4. **关于我区域**：个人介绍、技能展示、工作经历时间线
5. **联系区域**：联系信息和表单
6. **项目详情模态框**：点击作品查看详细信息

## 优化特性

- 图片懒加载提升性能
- 平滑滚动和过渡动画
- 响应式设计适配各种设备
- 无障碍访问支持

## 本地运行

直接在浏览器中打开 `index.html` 文件即可查看网站。
